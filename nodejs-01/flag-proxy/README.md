# flag-proxy acts as a Unleash Proxy

## How to install modules

```bash
yarn install
rm -rf node_modules
yarn install --offline
```

## How to start

```bash
yarn start
```

## Proxy Secerts

- `proxy-secret-key-1`
- `proxy-secret-key-2`

## Connection Information

- <http//localhost:4000/proxy>
